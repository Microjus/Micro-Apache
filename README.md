![Banner]

----
### Instalação
**Clone o Repositório**
```bash
$ git clone https://github.com/Microjus/Micro-Apache.git
```
### Execução
```bash
$ chmod +x micro-apache.sh
$ bash micro-apache.sh ou ./micro-apache.sh
```
